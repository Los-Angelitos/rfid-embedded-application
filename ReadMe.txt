# RFID Door Lock con ESP32 y Wokwi

Este proyecto simula una cerradura inteligente con ESP32 y lector RFID.  
Al escanear una tarjeta autorizada, se activa un servo para abrir la puerta.

## Simulación en Wokwi
Haz clic para ver la simulación en Wokwi:  
[🔗 Ver simulación](https://wokwi.com/projects/433613195311893505)

## Componentes usados
- ESP32
- Lector RFID (RC522)
- Servo motor
- Conexión WiFi
- API para validación remota de tarjetas
